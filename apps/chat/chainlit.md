# Welcome to nbadb

AI-powered NBA analytics — **160+ tables** of basketball data at your fingertips.

### What I can do

| Capability | Examples |
|---|---|
| **Query & Explore** | Standings, box scores, tracking data, shot charts |
| **Visualize** | Interactive Plotly charts, matplotlib court diagrams |
| **Analyze** | TS%, eFG%, Usage Rate, Net Rating, custom metrics |
| **Export** | CSV, XLSX, JSON, editable spreadsheets |
| **Iterate** | Refine results: "now filter to under 25", "add TS% column" |
| **Share** | Embeddable charts, branded social cards, thread summaries |
| **Save** | Export code, save analysis templates for reuse |
| **Research** | Latest NBA news, injury reports, trade context |

### Quick tips

- Use the **gear icon** to switch providers and models
- Pick a **profile** above for tailored analysis modes
- Results include **one-click export** buttons (CSV, XLSX, JSON, Spreadsheet)
- Say **"filter this"** or **"add a column"** to refine previous results
- Say **"make a social card"** or **"embed this chart"** for shareable output
- Say **"save as template"** to reuse an analysis pattern later
