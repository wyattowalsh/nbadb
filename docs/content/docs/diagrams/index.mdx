# Diagrams

The diagrams section is the visual film room for nbadb: whiteboard views for schema shape, playbook views for pipeline movement, and shot-chart overlays for endpoint coverage.

> **Film room note:** Pick the board that matches your question, then use the companion pages to zoom from fast orientation into generator-backed detail.

## Pick the Right Board

| Lens | Page | Best for |
| ---- | ---- | -------- |
| Whiteboard | [ER Diagram](/docs/diagrams/er-diagram) | Understanding how the core dimensions, facts, and bridges fit together |
| Generator board | [Auto ER Diagram](/docs/diagrams/er-auto) | Inspecting the full schema-derived entity inventory without curation |
| Playbook | [Pipeline Flow](/docs/diagrams/pipeline-flow) | Tracing how raw NBA API extracts become analytical outputs |
| Shot-chart overlay | [Endpoint Map](/docs/diagrams/endpoint-map) | Mapping endpoint families to their landing tables |
