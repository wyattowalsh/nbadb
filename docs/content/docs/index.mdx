# Getting Started

nbadb turns the NBA stats surface into a **public analytical model** — DuckDB staging, SQL-first transforms, and exports to DuckDB, SQLite, Parquet, and CSV.

## First five minutes

```bash
pip install nbadb
nbadb download       # pull the latest published dataset
nbadb schema         # confirm tables are visible
```

<Callout type="info">
  `nbadb download` pulls the latest published Kaggle dataset. For a full
  historical build from source, use `nbadb init` instead (takes hours, not
  minutes). Once seeded, `nbadb daily` is the standard game-day refresh.
</Callout>
