# Field Reference

Complete column-level documentation for all tables in the nbadb warehouse. Columns are extracted from Pandera schema definitions.

## Dimension Tables

### dim_player

| Column | Type | Nullable | Description |
|--------|------|:--------:|-------------|
| player_sk | int | No | Surrogate key for SCD2 player |
| player_id | int | No | Natural key -- NBA player identifier |
| full_name | str | No | Player full name |
| position | str | Yes | Player position |
| team_id | int | Yes | Team identifier (FK: dim_team.team_id) |
| jersey_number | str | Yes | Jersey number |
| height | str | Yes | Player height |
| weight | float | Yes | Player weight in lbs |
| birth_date | str | Yes | Date of birth |
| country | str | Yes | Country of origin |
| draft_year | int | Yes | Draft year |
| draft_round | int | Yes | Draft round |
| draft_number | int | Yes | Draft pick number |
| college_id | int | Yes | College identifier (FK: dim_college.college_id) |
| valid_from | str | Yes | SCD2 valid-from season |
| valid_to | str | Yes | SCD2 valid-to season |
| is_current | bool | No | Whether this is the current record |
