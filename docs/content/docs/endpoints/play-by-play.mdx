# Play-by-Play Endpoints

Three endpoints provide event-level game data.

## PlayByPlayV3

The latest play-by-play format with structured action types and shot coordinates.

- **Result sets**: `PlayByPlay`
- **Parameters**: `game_id` (required)
- **Key columns**: `action_number`, `clock`, `period`, `action_type`, `sub_type`, `description`, `shot_distance`, `shot_result`, `x_legacy`, `y_legacy`
- **Notes**: Preferred over V2 for games from 2021-22 onward

## PlayByPlayV2

Legacy play-by-play format with three-player event structure.

- **Result sets**: `PlayByPlay`, `AvailableVideo`
- **Parameters**: `game_id` (required)
- **Key columns**: `eventmsgtype`, `eventmsgactiontype`, `homedescription`, `visitordescription`, `player1_id`, `player2_id`, `player3_id`
- **Notes**: Used for historical games before 2021-22

## WinProbabilityPBP

Real-time win probability at each event in a game.

- **Result sets**: `WinProbPBP`, `GameInfo`
- **Parameters**: `game_id` (required)
- **Key columns**: `home_pct`, `visitor_pct`, `home_pts`, `visitor_pts`, `home_score_margin`, `period`, `seconds_remaining`
