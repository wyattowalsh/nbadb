# Page Types

## Purpose
Define the stable `kind` values used across maintained vault notes.

| kind | Use |
|------|-----|
| `overview` | Landing pages, route hubs, or MOCs |
| `concept` | Explanatory notes about architecture, tooling, or operations |
| `entity` | Notes centered on one concrete system, skill, or component |
| `comparison` | Side-by-side notes with stable dimensions |
| `source-summary` | Notes that summarize one source set and planned coverage |
| `index` | Navigation, inventory, source-map, or coverage pages |

## Current use
- `overview`: `wiki/index.md`, route pages
- `concept`: model, operations, tooling notes
- `entity`: analytics skill guide
- `source-summary`: `wiki/topics/*-source-summary.md`
- `index`: `indexes/*.md`
