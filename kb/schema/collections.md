# Collections

## Raw collections
| Collection | Purpose |
|------------|---------|
| `raw/sources/internal/` | Exact captures or stubs for repo-local canonical files when needed |
| `raw/sources/external/` | Exact captures or stubs for external docs, specs, and dataset pages |
| `raw/extracts/internal/` | Normalized manifests and grouped extracts from repo-local material |
| `raw/extracts/external/` | Normalized manifests and grouped extracts from external sources |
| `raw/assets/` | Local supporting assets |

## Wiki collections
| Collection | Purpose |
|------------|---------|
| `wiki/routes/` | Role or task-oriented route pages |
| `wiki/model/` | Warehouse model and lineage guidance |
| `wiki/operations/` | Operator runbooks |
| `wiki/tooling/` | Toolchain and stack notes |
| `wiki/topics/` | Topic, entity, and source-summary notes |

## Index collections
| Collection | Purpose |
|------------|---------|
| `indexes/*.md` | Navigation, source maps, coverage, and queues |
